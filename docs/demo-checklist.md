# MDT Hub Demo Checklist

## 1) 中英文切换即时生效
- 打开 `frontend.html`（或通过后端 `/` 访问）。
- 右上角点击 `EN`：
  - 顶部状态、按钮、主要面板标题、诊断标签应切换为英文。
  - 会诊室/分析视图标题应切换为英文。
  - 连接诊断区文案（API/models/agents/poll/WS）应切换为英文格式。
- 刷新页面后语言应保持为英文（localStorage 持久化）。
- 点击 `ZH` 后应立即切回中文，无需刷新。

## 2) 提交讨论与事件实时联通
- 在“会诊输入”填写病例 ID 与问题，点击“提交讨论”。
- 观察：
  - 顶部状态栏出现提交成功提示。
  - 会诊室 seat 气泡内容发生更新。
  - 实时时间线出现新增事件。
  - WebSocket 正常时状态为“已连接（实时）/connected”；异常时自动降级轮询并继续拉取事件。
- 点击“二轮互评”，确认时间线继续新增事件，冲突图可刷新。

## 3) Agent 模型配置可保存
- 在“Agent 模型配置”中选择任一 Agent 的模型，下拉后点击“保存/Save”。
- 页面提示应显示更新成功。
- 重新加载 Agent 列表后，所选模型应保持（持久化到后端）。
- 可再次切换其他模型并重复验证。
